Custom Interaction

Ragdoll uses the Unity Input System to handle inputs. Modders need to follow this convention when creating custom interactions.

Standard Weapon Interfaces (EntityFirableWeapon)

For most weapon inherit EntityFirableWeapon, you should implement the existing interfaces provided by the system. These handle the common interactions automatically:

Method VR Controller Flatscreen (Mouse/Keyboard)
OnTriggerPressed Trigger Button Left Mouse Button
OnTriggerReleased Trigger Release Left Mouse Release
OnAbilityPressed A/X Button R Key (Reload/Ability)
OnAbilityLongPressed Hold A/X Hold R

Using these interfaces ensures your weapon works consistently with the rest of the game's input handling.

Custom Input Handling

If you need more complex interaction, such as handling axis inputs (e.g., for controlling distance or rotation), you can directly interface with the GameplayInputs class.

Below is an example of how to implement custom axis binding

Example: Custom Axis Binding

This script demonstrates how to:

  1. Initialize the input system.
  2. Bind to the left or right hand weapon axis based on which hand is holding the object.
  3. Process the input value.
  4. Clean up listeners when the object is destroyed or unequipped.
using UnityEngine;
using UnityEngine.InputSystem;

public class CustomInteractionSample : MonoBehaviour
{
    private GameplayInputs inputs;
    private InputAction boundAxisAction;
    
    // Parameters to control the interaction
    private float currentValue;
    public float adjustSpeed = 1.0f;

    // Call this when the weapon/item is equipped to a hand
    public void BindAxis(bool isLeft)
    {
        // Initialize inputs if not already done
        inputs ??= new GameplayInputs();

        // Select the appropriate action based on handedness
        var nextAction = isLeft ? inputs.Player.WeaponAxisL : inputs.Player.WeaponAxisR;
        
        // Avoid re-binding if already bound to the same enabled action
        if (boundAxisAction == nextAction && nextAction.enabled)
            return;

        // Clean up previous binding if it exists
        if (boundAxisAction != null)
        {
            boundAxisAction.performed -= OnAxisUpdate;
            boundAxisAction.canceled -= OnAxisUpdate;
            boundAxisAction.Disable();
        }

        // Bind the new action
        boundAxisAction = nextAction;
        boundAxisAction.performed += OnAxisUpdate;
        boundAxisAction.canceled += OnAxisUpdate;
        boundAxisAction.Enable();
    }

    // Call this when unequipped
    public void UnbindAxis()
    {
        if (boundAxisAction != null)
        {
            boundAxisAction.performed -= OnAxisUpdate;
            boundAxisAction.canceled -= OnAxisUpdate;
            boundAxisAction.Disable();
            boundAxisAction = null;
        }
    }

    private void OnDestroy()
    {
        UnbindAxis();
        inputs?.Dispose();
        inputs = null;
    }

    // Callback for input events
    private void OnAxisUpdate(InputAction.CallbackContext context)
    {
        // Read the float value from the axis (e.g., joystick or touchpad)
        var value = context.ReadValue<float>();
        
        // Apply your logic here
        // For example, adjusting a target distance like in the Gravity Gun
        currentValue += Time.unscaledDeltaTime * adjustSpeed * value;
        
        Debug.Log($"Input Value: {value}, Current Value: {currentValue}");
    }
}
